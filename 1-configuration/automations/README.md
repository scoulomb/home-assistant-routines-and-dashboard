# Automations

Automations are classified by **functionality** (e.g., cinema, go-to-bed, play-random-music), not by remote or button. A single automation file may map multiple button subtypes from the same or different remotes.

:q> **Tip**: To use a Hue dimmer switch purely via HA automations, reset it in the Hue app and select "Configure in another app" — this prevents Hue from handling button presses and lets HA take full control.
