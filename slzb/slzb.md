## Setup ntwork adapter manually

See smligth installation guide: https://smlight.tech/manual/slzb-06/guide/installation/ (though they dot precise con generation here)

### Generate serial configuration

1. identify SLZB ethernet IP in your router interface (for example glInet: http://192.168.8.1/#/clients)
1. go there: http://192.168.8.192/z2m_zha
1. Generate yaml configuration

```yaml
# Pay attention, if you use the Z2M addon for HA, it is better to edit Z2M addon yaml configuration file directly (Select the 3 dot menu in the upper right corner → "Edit in YAML")
serial:
# Location of SLZB-MR3
  port: tcp://192.168.8.192:7638
  baudrate: 115200
  adapter: zstack
# Disable green led?
  disable_led: false
# Set output power to max 20
advanced:
  transmit_power: 20
```

### Use it in z2m add-ons

1. Go to home assistant Zigbee2mqtt add-on: http://homeassistant.local:8123/hassio/dashboard
1. Configuration, empty socat, mqtt and serial section
1. In serial section extract the following from previous generation (**remove serial keu**)

````shell
# Location of SLZB-MR3
port: tcp://192.168.8.192:7638
baudrate: 115200
adapter: zstack
# Disable green led?
disable_led: false
````

You will see socat section is re-generated by the way

### Note

1. Note if you do edit in yaml you can get the full conf

````yaml
data_path: /config/zigbee2mqtt
socat:
  enabled: false
  master: pty,raw,echo=0,link=/tmp/ttyZ2M,mode=777
  slave: tcp-listen:8485,keepalive,nodelay,reuseaddr,keepidle=1,keepintvl=1,keepcnt=5
  options: "-d -d"
  log: false
mqtt: {}
serial:
  port: tcp://192.168.8.192:7638
  baudrate: 115200
  adapter: zstack
  disable_led: false
````
And eventually add advanced feature.

At that stage we can see in logs that adapter is started and open web UI. Also here: http://192.168.8.192/z2m_zha, z2m/zha is activated

Tick show on sidebar for access to UI

### Choice of adapters

SLZB-MR3 offers both at http://192.168.8.192/z2m_zha
- EFR32MG24
  - adapter: ember
- CC2674P10
  adapter: zstack

ZHA (Zigbee Home Assitant) prefer embers (but should also work with zstack as older version supported: https://www.home-assistant.io/integrations/zha/#recommended-zigbee-radio-adapters-and-modules), Z2M prefers zstack but supports both https://www.zigbee2mqtt.io/guide/adapters/

I use Z2M with ZStack.
I excluded ZHA as sounds more limited device support as of 2025.
