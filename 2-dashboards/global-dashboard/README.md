- Separate views/tabs for each major room (Living Room, Bedroom, Bathroom, etc.) plus a Global tab

- Exclude all media players — media/audio is already handled by a separate [HiFi Remote Dashboard](../hifi-dashboard/hifi-dashboard.yaml)
- Include scenes (per room) as footer buttons for quick ambiance switching
- Smart plugs grouped by purpose (Charging, Mosquitoes)
- Quick global scenes (Go to Bed, Leave Home)
